Contact Form

This contact form is functional, you will get an error if you try to just open it.

Follow the steps below to make it work properly


  1. Install XAMPP using the default settings
  2. Activate Apache server and PhpMyAdmin
  3. Click on the "admin" button next to PhpMyAdmin
  4. Create a new database named "contact"
  5. Import the database from the database folder into your newly created database
  6. If you haven't changed the default settings of XAMPP, it should work, if not, check the database password/user in the file "send.php" and update them to your current settings.
